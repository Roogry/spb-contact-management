# Contact Management System
Creating RESTful API for Contact Management using Java Spring Boot based on youtube video [TUTORIAL SPRING BOOT RESTFUL API (BAHASA INDONESIA)](https://youtu.be/eFIBOVXilK4).

## Features
1. User Management
2. Contact Management
3. Address Management

### User API
* Register User
* Login User
* Update User
* Get User
* Logout User

### Contact API
* Create Contact
* Update Contact
* Get Contact
* List Contact
* Search Contact
* Remove Contact

### Address API
* Create Address
* Update Address
* Get Address
* List Address
* Remove Address
